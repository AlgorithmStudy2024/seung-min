## 정보

1. 지도는 가로 Ncm, 세로 Mcm 직사각형
2. 각 구역은 알파벳으로 되어있다. (정글 : J, 바다 : O, 얼음 : I)
3. 조사 대상 영역 K개
4. 각 영역에 정글, 바다 얼음이 몇 개 씩 있는지 구해라

## 입력

M N (1 <= M,N <= 1000)  
K (1 <= K <= 100000)  
M개의 줄 : 지도의 내용  
K개의 줄 : a b c d 왼쪽 위의 모서리 (a,b), 오른쪽 아래 모서리 좌표(c,d)

## 풀이

BruteForce : 1000억 (100000\*1000\*1000)  
누적 합 : M\*N  
(M+1)\*(N+1)사이즈의 2차원 배열 3개 사용
